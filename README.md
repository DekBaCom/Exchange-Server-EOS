# Exchange-Server-EOS

This is a Jekyll-based website providing information about the End of Support (EOS) for Microsoft Exchange Server.

## How to run locally

1.  **Prerequisites:** Ensure you have Ruby and Bundler installed.
2.  **Install dependencies:**
    ```bash
    bundle install
    ```
3.  **Build and serve:**
    ```bash
    bundle exec jekyll serve
    ```
4.  **View the site:** Open your browser to `http://localhost:4000`.

## Theme

This site uses the [jekyll-theme-primer](https://github.com/pages-themes/primer) theme.
